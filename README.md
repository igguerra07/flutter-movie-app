# MovieApp

MovieApp is in Flutter Version *3.9.2*

## Running the app

To get project dependencies:

```bash
flutter pub get
```

To generate files for DI (get_it):

```bash
flutter pub run build_runner build --delete-confliting-outputs
```
To run app:

```bash
flutter run
```
